# 🏥 Healthcare Analytics Dashboard (Power BI)

## 📊 Project Overview
This project presents a comprehensive Healthcare Analytics Dashboard built using Power BI. It analyzes hospital operations, patient demographics, doctor performance, treatment trends, and billing insights.

The dashboard provides interactive visualizations to support data-driven decision-making in healthcare management.

---

## 📁 Dataset Description

The project uses 5 datasets:

### 1. Patients
- Patient demographics and registration details

### 2. Doctors
- Doctor specialization, experience, and hospital branch

### 3. Appointments
- Appointment schedules, status, and visit reasons

### 4. Treatments
- Treatment type, cost, and date

### 5. Billing
- Payment details, revenue, and payment status

---

## 🎯 Project Objectives

- Analyze patient demographics and behavior  
- Evaluate doctor performance and workload  
- Monitor appointment trends and completion rates  
- Identify most common treatments  
- Track hospital revenue and payment status  
- Detect operational inefficiencies  

---

## 🧠 Key Features

- ✅ Advanced Data Modeling (Star Schema)
- ✅ Custom Date Table
- ✅ 40+ DAX Measures
- ✅ Time Intelligence (MoM Growth, Running Total)
- ✅ KPI Dashboard (Revenue, Patients, Appointments)
- ✅ Drillthrough Pages (Patient & Doctor Details)
- ✅ Tooltip Reports
- ✅ Dynamic Titles & Conditional Formatting
- ✅ Multi-page Interactive Dashboard

---

## 📊 Dashboard Pages

1. **Executive Overview**
   - KPIs, Revenue trends, Payment insights  

2. **Patient Insights**
   - Age distribution, gender analysis, insurance patterns  

3. **Doctor Performance**
   - Appointments, revenue contribution, specialization  

4. **Appointment Analysis**
   - Status, time slots, weekday trends  

5. **Treatment Analysis**
   - Treatment types, cost trends  

6. **Billing & Revenue**
   - Payment status, revenue growth, collection rate  

---

## 📈 Key Insights

- Most patients fall into the **18–45 age group**
- **X-ray and MRI** are the most frequent treatments  
- Some doctors handle significantly more appointments  
- A portion of payments remains **pending**, affecting cash flow  
- Revenue shows **monthly growth trends**
